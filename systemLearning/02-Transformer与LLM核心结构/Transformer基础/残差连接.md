# 残差连接residual

> **所属章节**: [[Transformer基础]]
> **所属模块**: [[02-Transformer与LLM核心结构]]
> **难度**: 基础~中级

## 1. 一句话定义

**残差连接（residual connection，残差连接，skip connection，shortcut）** 把一层的输入直接"跨层加"到输出：$x_{l+1}=x_l+F(x_l)$，即输出 = 输入 + 残差函数 $F$；它为深网提供了一条**梯度与信息的"高速公路"**，是 ResNet 能训上百层、Transformer 能堆深的关键结构。

## 2. 为什么需要它（动机与背景）

深网的老大难：**网络越深，反而越难训**（degradation problem）。

- 不是过拟合（训练误差也高）。
- 也不是梯度消失（BN 已缓解部分）。
- 真因：**恒等映射对纯堆叠网络太难学**——层层非线性变换容易把表示"带偏"，网络难以学成"这一层什么都不做"（identity），于是深层反而比浅层差。

**残差连接的洞察**：与其让层学完整变换 $H(x)$，不如让它学**残差** $F(x)=H(x)-x$。学"0"（什么都不加）比学"identity"容易得多——把 $F$ 推向 0 即可。于是深层可轻易退化为浅层（不会更差），深网才训得动。

Transformer 借鉴此结构，每个子层（attention/FFN）都包残差，使百层 LLM 可训。
> [!note] 解答：每个子层都包残差的作用 + 白话理解
> 
> **白话**：残差就是给信号（前向的信息、反向的梯度）留一条**抄近路的直通道**。没有它，信号要穿过每一层的矩阵乘 + 非线性的重重"过滤"，走到深层就变弱/失真；有了它，原始输入可以直接"跨层加"到输出，深层随时能拿到浅层的原始信息。
> 
> **每个子层（attention、FFN）都包残差，五个作用**：
> 
> 1. **信息高速公路（前向）**：$x_{l+1}=x_l+F(x_l)$，原始 token 嵌入能一路累加到顶层。第 50 层 = 第 0 层嵌入 + 50 个子层的小修正，长程信息不丢。无残差时，嵌入被层层非线性反复变换，到深层"面目全非"。
> 2. **梯度高速公路（反向）**：由链式法则 $\dfrac{\partial L}{\partial x_l}=\dfrac{\partial L}{\partial x_{l+1}}\left(1+\dfrac{\partial F}{\partial x_l}\right)$，那个 **"1"** 让梯度**绕过 $F$ 直达任意浅层**，不随层数连乘衰减 → 百层 LLM 也能训。这是深网可训的物理根。
> 3. **让深网能学"什么都不做"**：子层学的是残差 $F=H-x$，把它推向 0 这层就退化成 identity（跳过）。纯堆叠网络要精确拟合一组非线性的抵消才能实现 identity，极难 → 这就是深网退化（degradation）的根治。
> 4. **隐式集成**：展开 $x_L=x_0+\sum_{l}F_l$，等于 $L$ 条不同长度路径的加权集成（ResNet ≈ 浅网集成），带来稳定与泛化。
> 5. **与 LN 协同稳定数值**：残差保"梯度通路"，[[layer norm]] 保"数值范围"，缺一不可。现代 LLM 用 Pre-LN，主干 $x$ 不经 LN 直接沿残差链流，恒等 highway 更纯，深网更稳（见 §8.2）。
> 
> **无残差 vs 有残差**：
> 
> | 维度 | 无残差堆叠 | 每子层包残差 |
> |---|---|---|
> | 前向信息 | 层层变换，深层拿不到原始信号 | 输入一路累加，深层可引用早期信息 |
> | 反向梯度 | $\prod \partial F$ 连乘，易消失 | 含因子 1，不随层数衰减 |
> | 学恒等映射 | 需精确拟合非线性抵消，极难 | $F\to0$ 即跳过，极易 |
> | 深度上限 | 几十层就退化 | 几百层可训（ResNet/LLM） |
> 
> **一句话总结**：残差 = "输入直连 + 子层只学增量"，让信息/梯度跑得通、深网能退化成浅网、训练稳。它不是"加法技巧"，是深 Transformer 能堆到几十上百层的结构前提。
> 
> 详见本笔记 §3.2（梯度 highway）、§3.5（恒等映射）、§4（数学原理）。关联 [[layer norm]]、[[自注意力]]、[[FFN]]、[[decoder-only architecture]]。
## 3. 核心概念详解

### 3.1 基本形式

$$
x_{l+1}=x_l+F(x_l)
$$

- $x_l$：第 $l$ 层输入。
- $F$：子层（如 attention 或 FFN，常含 [[layer norm]]）。
- 加法是逐元素，要求 $F$ 输出与 $x_l$ **同形同维**。

### 3.2 梯度 highway

反向时由链式法则：

$$
\frac{\partial L}{\partial x_l}=\frac{\partial L}{\partial x_{l+1}}\left(1+\frac{\partial F}{\partial x_l}\right)
$$

关键在那个 **"1"**：梯度可以**不经 $F$ 直接回流**到任意浅层，不随层数衰减。于是几百层梯度也能传到首层 → 深网可训。这就是"highway"。

### 3.3 Transformer 里的残差

每个子层都包残差：

```
x → [SubLayer] → + → x'        # x' = x + SubLayer(x)
                 ↑
                 x（直连）
```

Transformer 一个 block = 两个残差子层（attention + FFN）：

$$
x' = x + \text{Attn}(\text{LN}(x))\quad(\text{Pre-LN})
$$
$$
x'' = x' + \text{FFN}(\text{LN}(x'))
$$

[[layer norm]] 的位置（Pre/Post）决定残差与归一化的顺序，影响稳定性（见 [[layer norm]]）。

### 3.4 维度匹配

$F$ 输出必须与 $x$ 同维才能相加。若要改变通道数（ResNet 下采样），用一个 **1×1 conv / linear** 把 $x$ 投影到匹配维再相加。Transformer 通常各层维度一致，无需投影。

### 3.5 残差与恒等映射

残差让网络**保留学恒等的能力**：把 $F$ 学成 0，则 $x_{l+1}=x_l$（这层什么都不做）。纯堆叠网络没有这种"短路"，学 identity 要精确拟合一堆非线性的抵消，极难。这是深网退化的根治。

## 4. 数学原理 / 公式

### 恒等梯度

$L$ 对 $x_l$ 的梯度含因子 $1$（来自 $x_{l+1}=x_l+F$ 对 $x_l$ 的偏导），即使 $\partial F/\partial x$ 很小，$1$ 仍在 → 梯度不消。对比无残差 $x_{l+1}=F(x_l)$，梯度 $\partial F/\partial x_l$ 连乘易消失。

### 残差 = 浅层集成（ensemble 视角）

展开 $L$ 层残差网络：

$$
x_L = x_0 + \sum_{l=0}^{L-1} F_l(x_l)
$$

输出是"初始 + 各层残差和"。可看作 $L$ 条不同长度路径的集成（ResNet = 浅网集成的隐式 ensemble），每条路径贡献一部分 → 集成带来稳定与泛化。

### 与 Highway / DenseNet 对比

- **Highway**：$x_{l+1}=T\cdot F(x_l)+(1-T)\cdot x_l$，有门控 $T$，更灵活但多参数。
- **DenseNet**：拼接 $[x_l, F(x_l)]$ 而非相加，特征复用但显存涨。
- **残差**：最简（加法、无参数、无门控），Transformer 用它。

## 5. 代码示例

```python
import torch
import torch.nn as nn

class ResidualBlock(nn.Module):
    def __init__(self, d, sublayer, dropout=0.0):
        super().__init__()
        self.sub = sublayer
        self.norm = nn.LayerNorm(d)
        self.drop = nn.Dropout(dropout)
    def forward(self, x):
        # Pre-LN 残差：x + Sublayer(LN(x))
        return x + self.drop(self.sub(self.norm(x)))

d = 512
attn = nn.MultiheadAttention(d, 8, batch_first=True)
ffn = nn.Sequential(nn.Linear(d, 4*d), nn.GELU(), nn.Linear(4*d, d))

x = torch.randn(2, 16, d)
# Transformer block = 残差(attention) + 残差(FFN)
x = ResidualBlock(d, lambda u, x=x: attn(u,u,u, need_weights=False)[0])(x)
x = ResidualBlock(d, ffn)(x)
print(x.shape)   # (2,16,512) —— 维度不变，可堆任意层

# 验证梯度 highway：深网梯度仍到首层
deep = nn.Sequential(*[ResidualBlock(d, nn.Identity()) for _ in range(200)])
x = torch.randn(1, 4, d, requires_grad=True)
deep(x).sum().backward()
print(x.grad.abs().mean().item())   # 仍非 0，梯度没消失
```

## 6. 与其他知识点的关系

- **上游（依赖）**: [[自注意力]]/[[多头注意力]]、[[FFN]]（残差包裹的子层）。
- **下游（应用）**: [[layer norm]]（残差内归一化）、[[decoder-only architecture]]（每 block 两残差）、训练稳定性（残差是深 LLM 可训的前提）。
- **对比 / 易混**:
  - **残差 vs 无残差堆叠**：残差让深网可训，无残差深网退化。
  - **残差 vs Highway/DenseNet**：残差最简加法；Highway 有门控；DenseNet 拼接复用。
  - **残差 vs 跳跃连接**：残差是加法 skip；某些 skip 是拼接（U-Net），不等价。

## 7. 常见误区与易错点

> [!warning] 误区清单
> 1. **维度不匹配直接加**：$F$ 输出与 $x$ 维度不同会报错，需投影层对齐。
> 2. **把残差当 ensemble 滥用**："残差=集成"是理论视角，不是让你显式建多个网络。
> 3. **残差里放改变形状的层**：残差子层应保持维度，改变形状会破坏"加回"。
> 4. **Pre-LN/Post-LN 混用**：残差与 LN 顺序不同稳定性差很大，见 [[layer norm]]，别乱放。
> 5. **以为残差解决一切不稳定**：残差防梯度消失，但不防梯度爆炸/数值溢出，仍需 [[梯度裁剪]]、[[mixed precision training]]。
> 6. **推理时残差还在**：残差是结构，训练推理都有，别 eval 时误删。

## 8. 延伸细节

### 8.1 残差与深度

残差让"有效深度"可变：某些层 $F\to0$ 等效跳过。LLM 实测有"skip-like"行为，部分层贡献小可剪（layer drop）。

### 8.2 Pre-LN 残差的稳定性

Pre-LN（先 LN 再子层再加残差）下，主干 $x$ 不经 LN 直接流过残差链 → 恒等 highway 更纯 → 深网更稳。Post-LN 主干每层被 LN 作用，深网易不稳。现代 LLM 全用 Pre-LN/RMSNorm。

### 8.3 残差与表示流

残差让初始嵌入一路加到顶层，模型可在任意层"引用"早期信息。这对长程依赖、信息保留有利，也使 logits 可追溯到初始 token（与 attention sink 现象相关）。

### 8.4 残差的显存代价

前向要存每层 $x$ 用于反向，层数多则激活显存大 → [[gradient-checkpointing]] 重算缓解。残差不增参数但增激活。

### 8.5 与 normalization 的协同

残差保"梯度通路"，LN 保"数值范围"，两者协同让深 Transformer 稳定。缺残差则梯度消失，缺 LN 则数值漂移/爆炸。
